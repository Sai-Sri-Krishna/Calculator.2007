# 🧮 Mini Calculator

Most calculators break when you type `1 + + 2`. I wanted to build one that doesn't. 

This is a clean, focused web calculator built to sharpen my skills in **JavaScript** and **responsive UI design**. It’s designed to feel less like a "web page" and more like a native app.

---

## ✨ Why this version?

Instead of just following a basic tutorial, I added features that make a real difference in daily use:

* **Keyboard First:** I mapped out the event listeners so you can use your numpad or laptop keys naturally. 
* **Logic Safety:** I implemented checks to prevent consecutive operators and fixed the classic `0.1 + 0.2` floating-point math bug that usually trips up JavaScript.
* **Persistent State:** If you accidentally refresh your tab, your current calculation stays right where you left it thanks to `localStorage`.
* **Responsive Layout:** Used CSS Grid to ensure the buttons feel tactile and usable on everything from a small mobile screen to a desktop.

---

## 🛠️ Built With

* **Structure:** Semantic HTML5
* **Styling:** CSS3 (Custom properties & Grid)
* **Brain:** Modern JavaScript (ES6 logic)
* **Persistence:** Web Storage API

---

## ⌨️ How to use it

You can click the buttons, or just use your keyboard:
- **Numbers/Operators:** Just type them!
- **Calculate:** Press `Enter` or `=`
- **Clear:** Hit `Esc`
- **Correction:** `Backspace` works exactly as you'd expect.
